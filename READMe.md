# Sovereign AI Unlimited - GitHub Codespaces Deployment

## Quick Start

1. **Fork this repository** to your GitHub account

2. **Open in Codespaces:**
   - Go to your forked repository
   - Click "Code" → "Codespaces" → "Create codespace on main"

3. **Auto-Start:**
   - The system automatically starts when Codespaces launches
   - Monitoring dashboard: http://localhost:3000

4. **24/7 Operation:**
   - System runs continuously in background
   - Auto-deploys successful jobs to real world
   - State is automatically saved and persisted

## Features

- ✅ 24/7 autonomous operation
- ✅ Real-time monitoring dashboard
- ✅ Auto-deployment to Ethereum mainnet
- ✅ State persistence and backup
- ✅ GitHub integration for auto-commits

## Access Points

- **Main Dashboard**: http://localhost:3000
- **API Status**: http://localhost:3000/api/status
- **Local Blockchain**: http://localhost:8545
